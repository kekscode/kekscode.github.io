Top 1 top tipp on Linux

My personal top #1 tip for using the ubiquitous command line utility top is to press i when you have launched it. It will help you to see much more than before by eliminating idle processes. Pressing i helps a lot, because most of the time you start top to monitor processes which are putting (a lot) of load on your system and not for watching idle processes doing nothing.

Awesome lists

Do you know what an awesome-list is? I am not talking about a shopping list, the movie Schindler's List or a piece of code with an array consisting of the names from all pokémon. I am talking about a best practice for naming community-curated lists of valuable things on the internet. If this sounds like a bookmark list to you, you're right. It is the same as a well-kept bookmark list for a certain topics, but with more than only one contributor and usually a strict policy for contributions to ensure constant quality.

Compare two or more remote files via SSH in vimdiff

A nice way to compare two (or more) files on different servers with vimdiff - which is just an alias for vim -d file1 file2 filex - works like that: $> vimdiff scp://remotehost//path/to/file1 scp://remotehost//path/to/file2 scp://remotehost//path/to/file3 Of course you can also compare files on localhost: $> vimdiff ./file1 scp://remotehost//path/to/file2 scp://remotehost//path/to/file3 Actually vimdiff is also a great difftool in tandem with git and other (D)VCS when you like to use the command line a lot.

3D printed safety razor

About this posting I'm into safety razors and i'm also enthusiastic about 3D printing. A few days ago i stumbled upon this model of a safety razor from a user called H3R0M4T3 and printed it using ordinary white PLA and not expecting too much. The result after my first shave: I like this homebrew razor very much. It gives a very close shave and the razor itself can be positioned in various ways along the plate before it is compressed by the top cap which gives you either the same blade configuration on both or a quite aggressive blade on one side and a more gently blade on the other side.
